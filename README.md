# AIgnite Social

AI-powered social media automation platform for boutique hotels.

AIgnite helps hotels create, customize, schedule, and publish social media content without needing a dedicated social media team. The platform uses AI to generate engaging posts and images for different occasions and events.

## Features

- AI-generated social media posts
- AI-powered image generation
- Post scheduling
- Content customization
- Hotel branding support
- Social media integrations

## Tech Stack

### Frontend

- React
- TypeScript
- Vite

### Backend

- Node.js
- Express.js

### Database

- MongoDB
